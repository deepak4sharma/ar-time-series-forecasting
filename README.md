# AR Time Series Forecasting Project

This project demonstrates how to use the Autoregressive (AR) model for time series forecasting.

## Objective
- Check stationarity
- Apply transformations
- Build AR model
- Forecast variables

## Tools Used
- Python
- Pandas
- Statsmodels
- Matplotlib

## How to Run
1. Install required libraries
2. Open the notebook
3. Run all cells
